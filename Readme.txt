How to run the Crime Reporting System (CRS) Project

1. Download the  zip file

2. Extract the file and copy cms folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name crsdatabase

6. Import crsdatabase.sql file(given inside the zip package in SQL file folder)

7.Run the script http://localhost/CRS (frontend)

8. For admin panel

http://localhost/CRS/admin  (admin panel)
